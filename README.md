# Glorified To-Do

A *not-so-ordinary* to-do list application designed to keep you on track with your day-to-day work — because **your only competition is yourself**.

This isn’t just about tasks.  
It’s about focus.  
It’s about consistency.  
It’s about making sure *you* are ahead of yesterday’s you.

---

## ✨ Features

- **Real-time timer** ⏱ — Track how long you’ve been at it.
- **Task management** ✅ — Add, check off, and edit your tasks with ease.
- **Quick keybinds** 🎯 — Perform actions without touching your mouse:
  - Start/Pause Timer → `Shift + Space`
  - Restart Timer → `Shift + R`
  - Input a new To-Do → `Shift + Enter`
- **Light glass-styled UI elements** 🪟 — For a clean, modern feel.
- **Inline editing** ✏️ — Update your tasks without breaking your flow.
